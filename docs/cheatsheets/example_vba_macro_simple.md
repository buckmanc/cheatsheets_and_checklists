# Example VBA Macro
Simple Version

1. Download the [example macro](/misc/ExampleMacro.xlsm)
1. Open it
1. Deal with any macro warnings
    - If you get the warning "Microsoft has blocked macros from running"
	1. Close the file
	1. Right click the file and check the box under Properties > General > Unblock
	1. Reopen the file
    - Click "Enable Content" if prompted
1. Click the "Run" button on the spreadsheet to see it work
1. To view/edit the code
    1. Right click the "Run" button
    1. Click "Assign Macro"
    1. Click "Edit"
1. Tweak to fit your needs
